# Roadmap

Each section below maps to either a written spec (linked) or a future area not yet specced.

## 1. Core Tracking

Foundation: habit tracking, journaling, and progress viewing with seamless daily data entry. Form view + spreadsheet view, multi-modal input.

→ [sections/core-tracking.md](sections/core-tracking.md)

## 2. Onboarding

Personalized flow that sets up each user's habit + journaling system without manual configuration. User exits onboarding ready to log Day 1.

→ [sections/onboarding.md](sections/onboarding.md)

## 3. AI Voice

Voice-first AI interaction for hands-free logging, coaching, and natural conversation across all surfaces where voice is relevant.

→ Implementation context: the `voice-commands` skill (Cartesia REST STT + GPT-4o-mini NLU pipeline) and the `voice-assistant` skill (Cartesia Line realtime coaching agent — current build).

## 4. Focus & Accountability

Aggressive blocking to enforce focus time, block distracting apps/sites, and push users toward consistency through proactive intervention.

## 5. Analytics & Reporting

Advanced visualization, natural-language query, trend charts, goal setting and tracking, comprehensive reporting.

## 6. Screen Time Integration

Import and analyze iOS Screen Time + Android Digital Wellbeing data to correlate digital habits with personal-growth metrics.

## 7. Wearable Integration

Connect Oura, Apple Watch, Fitbit, Garmin, etc. — pull biometrics and correlate with habit performance.

## 8. Data Import & Historical Analysis

Import from prior trackers (CSV, JSON, screenshots, Sheets), AI trend analysis on historical data, personalized recommendations.

## 9. Coaching Platform

B2B platform for coaches: content delivery, task/exercise assignment, client progress dashboards, and full visibility into client data.

## 10. Feedback Loop

Reinforcement and recovery: completion confirmation, streaks/milestone cues, weekly reviews, pattern-based nudges.

→ [sections/feedback-loop.md](sections/feedback-loop.md)

## 11. Data Visibility

Trust-building visibility into habits and reflections — dashboards, trends, comparisons, exports.

→ [sections/data-visibility.md](sections/data-visibility.md)

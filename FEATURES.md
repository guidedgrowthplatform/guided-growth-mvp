# Life Growth Tracker - Features

## Current Features
_Features that are currently implemented_

✅ Basic KPI configuration page
✅ Capture page with date picker
✅ Report page with calendar view
✅ LocalStorage persistence
✅ Sample metrics on first load
✅ Left sidebar navigation
✅ Frequency support (daily, weekdays, weekends, weekly)
✅ Input types (binary, short text, numeric)

## Future Features
_Brainstorming ideas for future features - add your ideas here!_

- **Screen Time Integration**: Connect and import screen time data from devices (iOS Screen Time, Android Digital Wellbeing, etc.)
- **Smart Device Data Integration**: Integrate data from smart devices and wearables:
  - Oura Ring data (sleep, activity, recovery metrics)
  - Smart watches (Apple Watch, Fitbit, Garmin, etc.)
  - Other health and fitness trackers
  - Import as much data as possible from available APIs
- **Data Import & Historical Analysis**:
  - Import data from any previous habit tracker (CSV, JSON, or other formats)
  - Bulk import historical data to populate past entries
  - AI-based trend analysis on historical data
  - Personalized recommendations based on patterns and trends
  - Visual trend charts and insights

- **External Data Visualization & Query**:
  - Import and visualize data from other tracking apps/services
  - Support for various data formats and sources
  - Interactive data visualization tools
  - Natural language query interface - ask questions about your data
  - AI-powered insights and answers based on imported tracking data
  - Cross-platform data integration
- **Growth Points (Gamification Layer)**:
  - Points awarded for every action in the app (habits, journaling, reviews, configuration, etc.)
  - Streak multipliers, milestones, and progress tiers
  - Visible point balance and progression throughout the UI
  - Tunable rules for points per action and cooldowns
- **PQ-Style Meditations & Micro-Practices**:
  - Guided micro-meditations modeled after Positive Intelligence PQ reps
  - Points awarded for completing practices
  - Reminders and prompts tied to focus/recovery moments

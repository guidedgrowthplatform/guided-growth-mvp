# Life Growth Tracker

A sleek, minimalist web application for tracking daily habits and personal metrics. Built with React, Vite, and Tailwind CSS.

## Features

- 📝 **KPI Configuration**: Create and manage custom metrics with different input types (binary, numeric, short text)
- 📊 **Capture Data**: Record your daily/weekly habits in form view or interactive spreadsheet view
- 📈 **Report & Trends**: Visualize your progress with a calendar-style spreadsheet view
- 🎨 **Futuristic Design**: Beautiful light blue glassmorphic UI
- 💾 **Local Storage**: All data stored locally in your browser

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <your-repo-url>
cd life-growth-tracker
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`)

## Usage

1. **Configure Metrics**: Go to the Configure page to set up your habits and KPIs
2. **Capture Data**: Use the Capture page to log your daily/weekly data
   - Toggle between Form View and Spreadsheet View
   - Edit historical data in spreadsheet view
3. **View Reports**: Check the Report page to see your progress in a calendar format
